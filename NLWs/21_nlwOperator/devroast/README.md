# 🚀 DevRoast

O **DevRoast** é uma ferramenta criada para quem não tem medo de levar a verdade na cara. Cole seu código, receba uma avaliação brutalmente honesta e veja onde você se encaixa no ranking da "vergonha" da programação.

## ✨ Funcionalidades

- **Code Roasting:** Análise automática de código com feedbacks sarcásticos e diretos.
- **Score de Qualidade:** Receba uma nota baseada na qualidade (ou falta dela) do seu código.
- **Shame Leaderboard:** Um ranking global dos códigos mais "criminosos" da internet.
- **Visual Tech:** Interface inspirada em editores de código modernos e terminais.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído com as tecnologias mais modernas do ecossistema web:

- **Next.js** (App Router)
- **Tailwind CSS v4** & **Tailwind Variants**
- **Shiki** (Realce de sintaxe com tema Vesper)
- **Base UI** (Primitivos de interface)
- **Drizzle ORM** & **PostgreSQL** (Persistência de dados)
- **Biome** (Linting e formatação de código)

## 🛠️ Ferramentas de Desenvolvimento

- **opencode** com MCPs **Pencil** (estudo de design) e **Context7** (consulta de documentação) para aceleração de engenharia.
- **Playwright** para testes de ponta a ponta (E2E).
